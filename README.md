# CrimeProject
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.2.0.
All the project is available on : https://github.com/coco83algo/Crime-Project


## Node modules
Run `npm i` to generate all the basic node modules required for this app.
Run `ng add @angular/material` to add Angular material to this project.
Run `npm install @swimlane/ngx-charts --save` to add Charts that are used in crimes part of this project.


## Map
To use correctly the map we added to this project, do the following steps:
  From inside the project directory, you should add the dependencies by using npm
    - Run `npm install leaflet`
    - Run `install @types/leaflet`
    - Run `install @asymmetrik/ngx-leaflet`


## Development server
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
